 Save and load player stats
 Make a program that:
Creates a player dictionary with:
name (string)
level (integer, start at 1)
health (integer, start at 100)
Saves all stats to a file called player.txt
Loads and prints the stats
example 
=== SAVING PLAYER ===
Name: Leo
Level: 1
Health: 100
Saved!

=== LOADING PLAYER ===
Name: Leo
Level: 1
Health: 100
Loaded!

Requirements:
Use dictionary for player
Write each value on its own line
Read and convert back to correct types
Use with open() for both operations
