# rember 

When printing a string in Python, you cannot directly add variables that contain int or float values to a string. You must either:
Convert the number to a string using str()
or
Use an f-string (f"...") to include variables easily.
Examples:
# Using str()
print("Total: " + str(total) + " kr")
# Using f-string (recommended)
print(f"Total: {total} kr") # so use can add your variables into string use f"..." and with variables name {}.
